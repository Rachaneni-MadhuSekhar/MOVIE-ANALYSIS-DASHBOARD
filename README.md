📌 Overview
This project is a Movies Data Analysis Dashboard built using Power BI. It provides insights into movie performance, genres, languages, ratings, and revenue trends using interactive visualizations.

📊 Key Insights
Total Movies: 9,598
Total Genres: 19
Average Rating: 6.36
Total Languages: 37
Average Runtime: 99.70 mins
Average Revenue: 58.90M
Average Vote Count: 1.36K

📈 Dashboard Features
🔝 Top Analysis
Top 10 Movies by Revenue
Top Genres by Revenue

📅 Trend Analysis
Popularity of Movies Over Time (Year-wise trend)

🌍 Language Insights
Top 5 Languages by Number of Movies
🎛️ Filters (Slicers)
Year Filter
Original Language Filter
